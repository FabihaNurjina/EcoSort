Fabiha's Machine Learning Model - Image Edition 📷

EcoSort:
EcoSort is an image classification model designed to identify whether everyday items are biodegradable or recyclable. This project aims to raise awareness about sustainable practices and make recycling easier for individuals by providing a simple AI-based tool that encourages proper waste sorting and environmental responsibility.
